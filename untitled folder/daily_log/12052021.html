Researched a bit on the issue with fetching from VueJS. I was having a problem with a JSON parse error and I wasn't quite sure why. My immediate first thought was that the JSON on the MongoDB was incorrect but I triple checked and it was all fine. Ok, so the issue must be issue that the response is not being sent correctly from ExpressJS? Not that either, I've double checked with Postman and simply on a normal browser. So, after going backwards and forwards I can confirm the problem is that the fetch request from within Vue is not actually reaching the local host for the ExpressJS server. Ugghh!!! Here we go again, different ports and their incompatibility I thought I had been there and done that but I clearly didn't understood it well enough when I did it so I'll have a proper look tomorrow. I must admit I am a bit fed up with VueJS.
